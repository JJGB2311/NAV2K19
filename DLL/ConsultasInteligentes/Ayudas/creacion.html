AYUDA PARA LA CREACION DE CONSULTAS INTELIGENTES

I. Botones Generales usados por el Sistema:
i. Botones Agregar (flecha roja) : Los cuales tienen como función ir creando una cadena para la consulta final, dependiendo de lo que el usuario necesite.
ii. Botones Cancelar (flecha azul): En caso de que exista una equivocación en la consulta presionando este botón podrá limpiar los campos que se encuentran cerca.
iii. Botón Crear (flecha verde): Crea la consulta después de haber llenado los campos.
iv. Botón Eliminar (flecha naranja): Se encarga de limpiar todos los campos.


II. Posibles errores:
i. Campos vacíos: No debe dejar campos vacíos, ya que el sistema no lo dejará avanzar
